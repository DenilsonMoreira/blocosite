- Next.js support policy: https://nextjs.org/support-policy
- Next.js 16 docs: https://nextjs.org/docs
- Fastify latest docs: https://fastify.io/docs/latest/
- PostgreSQL current docs: https://www.postgresql.org/docs/current/
- Prisma system requirements: https://docs.prisma.io/docs/orm/reference/system-requirements
- Prisma supported databases: https://www.prisma.io/docs/orm/v6/reference/supported-databases
